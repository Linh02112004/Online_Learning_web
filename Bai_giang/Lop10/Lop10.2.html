Chương 2: Bất phương trình và hệ bất phương trình bậc nhất hai ẩn.


Tài liệu

Xem tài liệu tại đây.


Tóm tắt kiến thức:

I - Bất phương trình bậc nhất hai ẩn.

  1. Bất phương trình bậc nhất hai ẩn là bất phương trình có dạng:

    ax + by + c < 0, ax + by + c > 0, ax + by + c ≤ 0, ax + by + c ≥ 0

    Trong đó a, b, c là những số cho trước sao cho a² + b² ≠ 0; x và y là các ẩn

  2. Nghiệm của bất phương trình ax + by + c < 0 là một cặp số (x₀, y₀) sao cho ax₀ + by₀ + c < 0

II - Biểu diễn tập nghiệm của Bất phương trình bậc nhất hai ẩn

  1. Miền nghiệm của bất phương trình
  2. Cách xác định miền nghiệm

  2. Xác định được miền nghiệm của hệ bất phương trình
  3. Nhận biết hệ bất phương trình.
  4. Biết biểu diễn miền nghiệm của hệ.
